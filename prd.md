# 📑 Minimal Fit v1.56 PRD (상세 설계서)

## 1. 제품 개요 (Product Overview)
* **제품명**: Minimal Fit (미니멀 핏)
* **한 줄 정의**: 인체해부학적 정밀 타겟팅(대분류-소분류)과 AI 템포 엔진이 결합된 리워드형 지능형 운동 플랫폼.
* **핵심 목표**: AI 자동화 파이프라인을 통한 사용자 맞춤형 운동 환경 표준 모델 확립.

## 2. 타겟 유저 (Target Audience)
* **Heavy User**: 근육의 세부 타겟(상부, 하부 등)에 따른 정교한 종목 배분을 선호하는 사용자.
* **Returning User**: 저장된 루틴으로 빠르게 재개하고 리워드를 획득하려는 기존 사용자.
* **New User**: 최초 진입 시 숙련도 설정을 통해 맞춤형 루틴을 생성해야 하는 사용자.

## 3. 기능 상세 정의 (Feature Specifications)

### [핵심 엔진 및 로직]
* **[F1] Zero-Click 세트 동기화 (Set Sync)**: 
    - 1세트 수정 시 `isEdited: false`인 후속 세트에 실시간 전파.
    - 사용자가 개별 수정한 데이터는 보호함(Override 방지).
* **[F2] 1초 단위 AI 박자 가이드 (Tempo Engine)**: 1초 간격 한국어 음성 템포 가이드 제공.
* **[F20] 글로벌 운동 타이머**: 운동 시작 시점부터 종료까지 전체 소요 시간을 실시간 추적하여 노출.

### [데이터베이스 및 루틴]
* **[F15] 해부학적 정밀 DB (Anatomic Database)**:
    - 계층: 대분류(가슴/등/하체/어깨/팔/복근) -> 소분류(상/중/하 등) -> 종목.
    - **소분류 지점당 최소 3개 이상의 종목** 배치 및 운동 기구(바벨/덤벨/머신/맨몸) 구분.
* **[F21] 숙련도별 볼륨 자동 설계**: 
    - 초급: 15세트(5종목), 중급: 20세트(7종목), 고급: 30세트(10종목).
* **[F14] 온보딩 및 스마트 홈**: 
    - 신규 유저 강제 온보딩. 홈 화면 상단 대형 카드와 하단 50:50 그리드 버튼 배치.

### [수익화 및 리워드]
* **[F16] 일일 광고 리워드 시스템**: 
    - 휴식(광고) 시청 시 30FP 적립. 하루 최대 2회 한정.
* **[F17] 통합 리워드 샵**: 헤더에 포인트 현황과 샵 진입 버튼 통합. 가상 물품 구매 UI.

## 4. 데이터 및 시스템 설계 (System Design)
* **프론트엔드**: Next.js 기반 반응형 레이아웃 (중앙 프레임 480px 고정).
* **상태 관리**: `view`, `userPoints`, `rewardCountToday`, `elapsedTime` 전역 관리.
* **배포 정책**: GitHub Pages 배포 (`output: 'export'`, `basePath: '/workout_ads'`).

## 5. AI 자동화 파이프라인 적용 (Automation Pipeline)
* **Step 1 (기획)**: 본 PRD를 통한 기능 명세 자동 갱신.
* **Step 2 (검증/테스트)**: AI가 저장소 빌드 성공 여부를 상시 모니터링하여 무결성 확인.
* **Step 3 (개발)**: PRD 명세를 기반으로 AI가 코드 생성 및 최적화.
* **Step 4 (배포)**: GitHub Actions를 통해 정적 파일 배포 자동화.

## 6. 수익화 및 확장 계획 (Monetization)
* **광고 수익 분배**: 수익의 30%를 유저에게 FP로 적립 (7:3 배분).
* **프리미엄 모델**: 광고 제거 및 무제한 루틴 저장 기능 제공 예정.
* **커머스 확장**: 포인트 샵 내 실제 헬스 브랜드 입점 및 제휴.

## 7. 문서 관리 및 버전 관리 정책
* **버전 체계**: Semantic Versioning 적용.
* **PRD 제공 방식 [필수]**: 변경 시 항상 **전문(Full Text)**을 코드 블록 내에 작성하여 제공함.
* **커밋 메시지 표준**: `커밋명 [ver] 제목 / 내용 [commit 주요 내용]` (줄바꿈 준수)
