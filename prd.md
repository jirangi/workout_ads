# 📑 Minimal Fit v1.04 상세 설계서

## 1. 제품 개요 (Product Overview)
- **제품명**: Minimal Fit (미니멀 핏)
- **한 줄 정의**: AI 박자 가이드와 자동 동기화 로직을 통해 운동 중 조작을 0에 수렴하게 만드는 지능형 운동 보조 플랫폼.
- **핵심 목표**: 시장 조사부터 출시까지 이어지는 AI 자동화 파이프라인의 첫 번째 표준 모델 확립.

## 2. 타겟 유저 (Target Audience)
- **Heavy User**: Strong, Hevy 등 기존 앱의 반복적인 세트 입력에 피로감을 느끼는 중급 이상의 웨이트 트레이너.
- **Tempo Seeker**: 플랭크, 푸쉬업 등 정확한 박자와 템포 유지가 실력 향상에 직결되는 맨몸 운동 사용자.

## 3. 기능 상세 정의 (Feature Specifications)

### [F1] Zero-Click 세트 동기화 (Set Sync)
- **상세**: 첫 번째 세트에서 수정된 무게(Weight)와 횟수(Count) 데이터를 AI가 인식하여, 해당 종목의 남은 모든 세트에 즉시 복사 및 적용함.
- **가치**: 기존 사용자들이 느끼는 '매 세트 수동 복사'의 지루함을 완전히 해결함.

### [F2] 0.1s 단위 AI 박자 가이드 (Tempo Engine)
- **상세**: 설정된 주기에 맞춰 "One, Two" 음성을 실시간 출력함. 운동 중 0.1초 단위로 속도 조절이 가능하며, 남은 시간을 시각적으로 카운트다운함.
- **가치**: 단순 타이머 기능만 제공하는 앱들과 차별화된 '페이스 메이커' 역할 수행.

### [F3] 빅 스텝퍼 UI & 자동 확정 (Interaction)
- **상세**: 수치 영역 클릭 시 대형 상하 조절 버튼 노출. 별도의 확인 버튼 없이 외부 영역 터치(Blur) 시 데이터가 즉시 저장됨.
- **상단에 운동 모드 전환 탭(무게+횟수/횟수/시간)을 제공하여 유동적인 운동 입력을 지원함.**
- **가치**: 땀이 난 손이나 거친 호흡 중에도 정밀한 조작 없이 빠르게 값을 수정할 수 있음.

### [F4] 중급자용 스마트 증량 로직 (Progression)
- **상세**: 사용자 숙련도가 '중급'인 경우, 마지막 성공 데이터로부터 14일 경과 시 AI가 자동으로 무게 5kg 상향을 제안하고 반영함.
- **가치**: 유저가 직접 계산할 필요 없이 AI가 성장을 관리해주는 비즈니스 로직 탑재.

## 4. 데이터 및 시스템 설계 (System Design)
- **프론트엔드**: Next.js 기반 반응형 웹/앱 레이아웃 (중앙 프레임 480px 고정).
- **파일 아키텍처: 유지보수 및 AI 코딩 효율을 위해 layout.tsx(공통 프레임)와 page.tsx(비즈니스 로직)를 엄격히 분리하여 관리함.**
- **데이터 구조**: 운동 부위별(7대 대분류) 계층 구조를 가진 JSON 스키마 사용.
- **인프라**: GitHub Actions를 통한 자동 빌드 및 배포 시스템 연동.

## 5. AI 자동화 파이프라인 적용 (Automation Pipeline)
- **Step 1 (기획)**: 시장 분석 데이터를 프롬프트에 주입하여 본 PRD를 자동 갱신함.
- **Step 2 (개발)**: 본 PRD의 기능 명세를 기반으로 AI가 Next.js 컴포넌트와 비즈니스 로직 코드를 생성함.
- **Step 3 (테스트)**: PRD의 성공 조건(무게 동기화 여부 등)을 검증하는 자동 테스트 스크립트 실행.

## 6. 수익화 및 확장 계획 (Monetization)
- **광고**: 휴식 시간 진입 시 전면 광고(Interstitial) 송출로 수익 확보.
- **구독**: AI 자동 플래닝 및 상세 리포트 기능을 포함한 프리미엄 등급 운영.
