# 📑 Minimal Fit v1.34 PRD (상세 설계서)

## 1. 제품 개요 (Product Overview)
* **제품명**: Minimal Fit (미니멀 핏)
* **한 줄 정의**: AI 박자 가이드와 자동 동기화 로직을 통해 운동 중 조작을 0에 수렴하게 만드는 지능형 운동 보조 플랫폼.
* **핵심 목표**: 시장 조사부터 출시까지 이어지는 AI 자동화 파이프라인의 표준 모델 확립.

## 2. 타겟 유저 (Target Audience)
* **Heavy User**: 세트 입력 피로감을 느끼는 중급 이상의 트레이너.
* **Tempo Seeker**: 1초 단위 템포 유지가 중요한 사용자.
* **Mobile First**: 작은 화면에서도 한눈에 들어오는 UI를 선호하는 사용자.

## 3. 기능 상세 정의 (Feature Specifications)

### [핵심 엔진]
* **[F1] Zero-Click 세트 동기화 (Set Sync)**: 
    - 1세트 수정 시 `isEdited: false`인 모든 후속 세트에 즉시 전파.
    - 데이터 구조: `Array<{id, weight, reps, isEdited}>`.
* **[F2] 1초 단위 AI 박자 가이드 (Tempo Engine)**: Web Speech API 활용, 1초 간격 음성 카운트 및 시각적 애니메이션.
* **[F3] 빅 스텝퍼 UI (Interaction)**: 
    - 화면 중앙에 대형 수치 배치. 
    - 터치 타겟 극대화 및 시각적 피드백 강화.
* **[F13] 휴식 모드 (Rest UI)**: 
    - 세트 완료 시 화면 전체를 덮는 다크 모드 기반 휴식 타이머 노출.
    - SKIP 버튼 제공 및 타이머 종료 시 자동 다음 세트 전환.

### [루틴 및 데이터]
* **[F14] 다중 루틴 프리셋**: 초/중/고급 숙련도별 5종 루틴 제공.
* **[F15] 운동 메타데이터**: 가슴, 등, 어깨, 하체, 팔, 복근 대분류 및 상/중/하부 소분류 체계.

### [인프라]
* **[F6~F8] AI Quality Gate**: GitHub Actions 안정화 및 AI 주도적 무결성 검증 활성.

## 4. 데이터 및 시스템 설계 (System Design)
* **UI 레이아웃 [필수]**: 
    - **중앙 집중형 배치**: 모든 핵심 입력 UI는 화면의 가로/세로 정중앙(`flex items-center justify-center`)에 배치.
    - **모바일 최적화**: 너비 480px 고정 및 모바일 브라우저 100vh 대응.
* **파일 아키텍처**: `layout.tsx`(프레임), `page.tsx`(로직), `data/`(데이터 모듈) 분리.

## 5. AI 자동화 파이프라인 적용 (Automation Pipeline)
* **Step 1~4**: 기획-검증-개발-배포로 이어지는 자동화 프로세스 준수.
